---

## 1. 개요

AI의 독설과 게이미피케이션을 통해 사용자의 지출을 억제하고 자산을 보호하는 '초절약' 보조 앱.
지출을 입력하면 AI가 B급 감성의 킹받는 잔소리를 생성하고, 절약 성적에 따라 '거지 등급'이 변화하는 재미 요소로 자발적 절약을 유도한다.

---

## 2. 문제 정의

고물가 시대에 MZ세대는 가계부 앱을 설치하고도 3일 안에 포기한다.
기존 가계부 앱의 문제는 **지루함**과 **죄책감 없는 기록**이다.
지출을 입력해도 아무 반응이 없기 때문에 입력 동기가 사라진다.
'거지방' 문화처럼 절약을 놀이화하고, AI가 즉각적인 감정 반응(잔소리)을 제공하면
입력 행동 자체를 게임처럼 만들어 지속성을 높일 수 있다.

---

## 3. 타겟 유저

| 페르소나 | 설명 | 주요 시나리오 |
|----------|------|---------------|
| 주 사용자 | '거지방' 문화에 익숙한 2030 사회초년생. 절약을 놀이처럼 즐기고 싶음 | 카페 커피 값 입력 → AI 잔소리 → 친구에게 캡처 공유 |
| 부 사용자 | 가계부 작심삼일을 반복하는 금융 초보자. 고물가에 지출 통제 필요 | 예산 초과 알림 → 등급 하락 → 다시 절약 도전 |

---

## 4. 성공 지표 (KPI)

- 출시 후 한 달 내 사용자 100명 달성
- 유저당 주당 지출 입력 횟수 5회 이상
- AI 잔소리 결과 SNS 공유 발생 (바이럴 확인)

---

## 5. 기술 스택

- **프레임워크**: Next.js 15 (App Router)
- **스타일링**: Tailwind CSS
- **백엔드/DB**: Supabase (PostgreSQL + RLS + Auth 매직링크)
- **AI**: Gemini 2.5 Flash (확정 전 사용; 실제 최신 모델명 확인 필요)
- **언어**: TypeScript (strict mode)

---

## 6. 배포 환경

Vercel + Supabase Free Tier. 월 유지비 $0 목표.

---

## 7. MVP 핵심 기능

- [ ] **AI 등짝 스매싱**: 지출 금액/내역 입력 시 AI가 설정된 페르소나로 독설 피드백 생성
- [ ] **지출 대시보드**: 일간/주간 지출 합계 및 내역 리스트 조회
- [ ] **거지 등급 시스템**: 지출 억제 성적에 따른 캐릭터 등급 변화 및 배지 부여
- [ ] **수익화 기초**: 유료 페르소나 잠금 UI + "다음 업데이트 예약" 수요 검증 버튼
